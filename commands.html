Commands Reference

All build, deployment, and cleanup commands

Development Commands

npm run dev
Start development server at http://localhost:3000
npm run build
Build project for production
npm start
Start production server
npm run lint
Run ESLint to check code quality

Clean Installation (Linux/macOS)

rm -rf node_modules package-lock.json
Remove node_modules and lock file
npm cache clean --force
Clear npm cache completely
npm install
Fresh install all dependencies
node -v && npm -v
Verify Node.js and npm versions

Clean Installation (Windows)

del /s /q node_modules package-lock.json
Remove node_modules and lock file
npm cache clean --force
Clear npm cache completely
npm install
Fresh install all dependencies
node -v && npm -v
Verify Node.js and npm versions

Deployment Commands

vercel --prod
Deploy to Vercel production
vercel ls
List all Vercel deployments
git push origin main
Push to GitHub (triggers GitHub Actions)

Troubleshooting Commands

rm -rf .next
Remove Next.js build cache
npm list
List all installed dependencies
npm audit fix
Fix security vulnerabilities
npm update
Update all dependencies to latest versions
