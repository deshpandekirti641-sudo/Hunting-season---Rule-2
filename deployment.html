Deployment Guide

Complete setup and deployment instructions for Neon Nexus City

Prerequisites

  • Node.js v20.x or higher
  • npm v10.x or higher
  • Git for version control
  • A code editor (VS Code recommended)

Local Development Setup

Step 1: Clean Installation (Linux/macOS) 
rm -rf node_modules package-lock.json
npm cache clean --force
npm install
npm cache clean --force
node -v
npm -v
Step 1: Clean Installation (Windows) 
del /s /q node_modules package-lock.json
npm cache clean --force
npm install
npm cache clean --force
node -v
npm -v
Step 2: Run Development Server 
npm run dev

Visit http://localhost:3000 to play the game

Step 3: Build for Production 
npm run build
npm start

Deployment Platforms

Vercel (Recommended)

npm install -g vercel
vercel --prod

Render

Connect your GitHub repository and set build command to npm run build

Railway

Connect your GitHub repository, Railway will auto-detect Next.js

Heroku

heroku create your-app-name
git push heroku main

Environment Variables

No environment variables required for basic gameplay. Optional:

  • NEXT_PUBLIC_APP_NAME - Application name
  • NEXT_PUBLIC_APP_VERSION - Application version

Troubleshooting

Build fails with npm errors:

rm -rf node_modules package-lock.json .next
npm cache clean --force
npm install

Game doesn't load:

  • Check browser console for errors (F12)
  • Verify all dependencies: npm list
  • Restart dev server: npm run dev
